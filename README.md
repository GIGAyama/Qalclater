# Qalclater

**小学生向け計算ゲーム学習アプリ** - みんなで楽しく算数を学ぼう！

> React製のブラウザ計算ゲーム。1〜6年生の算数をカバーし、P2Pマルチプレイ・きせかえ・デイリーミッションなどゲーミフィケーション要素を搭載。

## デモ

https://gigayama.github.io/Qalclater/

## 特徴

### 3つのゲームモード

| モード | 説明 |
|--------|------|
| **スコアアタック** | 制限時間内にハイスコアを目指す（1〜10分） |
| **タイムアタック** | 全20問をできるだけ早くクリア |
| **サドンデス** | 1問でもまちがえたら終了 |

- 正解した瞬間に自動で次の問題へ進むスピーディーな操作感
- コンボシステム（5コンボ以上でフィーバーモード発動）
- 手書きメモキャンバス付き

### リアルタイムマルチプレイ

PeerJS（WebRTC）によるP2P通信で、サーバー不要のマルチプレイを実現。

- **へやをつくる** - 6桁のルーム番号を発行、QRコード・URL共有に対応
- **へやに入る** - ルーム番号を入力して参加
- リアルタイムスコア同期 & ライブランキング表示
- リーダー（ホスト）もメンバーと一緒にゲームをプレイ可能

### 50以上の計算コース（小学1〜6年）

学年別にプリセットされた計算問題コース群。

| 学年 | 主な内容 |
|------|----------|
| 1年 | あわせて10、くりあがり、ひきざん、とけい |
| 2年 | 九九（各段・穴埋め）、2けたの計算、分数 |
| 3年 | わり算、あまり、3〜4けた、小数、単位変換 |
| 4年 | 大きな数（億・兆）、がい数、面積、角度 |
| 5年 | 割合(%)、公倍数・公約数、3.14、体積 |
| 6年 | 分数かけわり、速さ、比、場合の数、データ |

#### コースのカスタマイズ

- オリジナル問題の作成・編集
- 共有コードで他のユーザーとコースを共有
- AI（ChatGPT / Claude）で問題を自動生成し、CSVインポート
- **にがて克服ボックス** - まちがえた問題を自動収集し重点練習

### ゲーミフィケーション

- **レベルシステム** - EXPを獲得してランクアップ（🌱かけだし → 👑計算マスター → 🌌計算神）
- **デイリーミッション** - 毎日3つのミッションをクリアしてコインを獲得
- **きせかえショップ** - コインでアバターパーツ（76種以上）やテーマ（13種）を購入
- **がんばりグラフ** - 直近7日間の学習状況を可視化

### アクセシビリティ

- 漢字にふりがな（ruby）対応
- 13種のカラーテーマ（ダーク・さくら・オーシャン・サイバーなど）
- 効果音 & BGM（Web Audio APIによる合成音）、ON/OFF切替
- モバイルファースト設計（タッチ操作・バイブレーション対応）
